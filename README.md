# Find consecutive primes of euler's constant

<br/>

### Setup
```
npm install
```

### Run
```
npm run start [Number of consecutives]
```

<br/>
<br/>

### Example

```
$ npm run start 12

> Euler-Prime-Numbers@1.0.0 start
> node index.js "12"

First 12 consecutive prime numbers in e:  355252327572
Found at 27448th decimal place

```
